Tables de multiplication

Pour un nombre donné, générer la table de multiplication correspondante

Dans un premier temps, le nombre correspondant à la table sera "hard codé" et affecté à une variable. Pour changer de table, il faudra donc ré affecter une nouvelle valeur à cette variable.

Fonctionnel mais pas très pratique...

Vous pouvez faire évoluer votre code et le rendre plus modulable en créant une fonction qui acceptera un paramètre (qui représentera le nombre pour lequel vous voulez afficher la table) et qui exécutera votre code en utilisant la valeur de ce paramètre (argument).

Et ça c'est bien plus cool !


function double(value) {
    let result = value * 2;
    console.log(result);
}

double(6); // affichera 12 dans la console

Détecteur de palindrome

Détecter si un mot est un palindrome.

Instant culture : un palidrome est un mot (ou un groupe de mots) que l'on peut lire indifféremment de gauche à droite ou de droite à gauche.

Dans un premier temps, le mot à tester sera "hard codé" et affecté à une variable. Pour tester un autre mot, il faudra donc ré affecter une nouvelle valeur à cette variable.

Comme pour l'exercice des tables de multiplication, c'est fonctionnel mais pas très pratique.

Faites évoluer votre code et rendez le plus modulable en créant une fonction qui acceptera un paramètre (le mot que vous voules tester) et qui exécutera votre code en utilisant la valeur de ce paramètre (argument).

On affichera en console le message "C'est un palindrome" si le mot en est un, et "Ce n'est pas un palindrome" dans le cas contraire.

Générateur de mot de passe AKA Boss Final

Créez un générateur de mot de passe

Au lancement du script, vous générerez un mot de passe de 8 caractères pouvant contenir des chiffres, des lettres minuscules, des lettres majuscules et les caractères spéciaux "#$%&".

Allez plus loin en créant une fonction qui prendra en paramètre la longueur du mot de passe. 


function generatePassword(length) {
    // code à exécuter
}

// appel de la fonction
generatePassword(12) // génère un mot de passe de 12 caractères

Allez encore plus loin. Adaptez votre fonction pour qu'on puisse lui passer en paramètres les options suivantes :

  • length
  • lower
  • upper
  • numbers
  • symbols

Vous vous assurerez que le mote de passe contient au moins 1 caractère correspondant à chaque paramètre qui est à true. 


function generatePassword(length, lower, upper, numbers, symbols) {
    // code à exécuter
}

// appel de la fonction
generatePassword(10, true, true, true, false) // génère un mot de passe de 10 caractères contenant au moins 1 lettre minuscule, 1 majuscule, 1 chiffre, pas de  symbole
